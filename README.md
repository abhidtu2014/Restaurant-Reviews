# Restaurant-Reviews
MERN Stack
M: MongoDB Atlas
E: Express
R: React
N: Node.js


This Project is inspired from Freecodecamp's MERN Stack Tutorial.

# Backend Data Flow

Express Router -> Service Controller -> DataBase Access Object (DAO)

# Frontend Data Flow

React Router -> React Component -> UseEffect() -> HTTP Middleware -> API Endpoints
